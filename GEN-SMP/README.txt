                           * Genjitsu СМП Меню, версия 2.0 *
---------------------------------------------------------------------------------------

Утилита для монтирования образов на модуле-эмуляторе, совместимом со стандартом
Genjitsu (STMP или PIMP).

Утилиту необходимо скопировать на модуль-эмулятор. Исходно программа имеет название
"AUTORUN.BIN", что позволяет модулю автоматически загружать ее при включении.
Данная программа выводит список файлов в памяти модуля-эмулятора и предлагает
пользователю выбрать образ для монтирования.

Согласно стандарту Genjitsu файл должен иметь имя в формате '8.3'.

Новая версия предлагает удобное меню выбора файла вместо ручного ввода.
Стрелки влево и вправо позволяют выбрать страницу. Стрелки вверх и вниз - файл.
Исключена возможность создания нового файла, поскольку ни один из модулей-эмуляторов
на сегодняшний день не поддерживает режим записи.

После подтверждения файла нажатием клавиши ВК, произойдет его монтирование.
Теперь можно пользоваться модулем-эмулятором как обычным ПЗУ модулем с
запрограммированными данными.

Для сброса образа нужно отсоединить модуль от микрокомпьютера или воспользоваться
программой "Gen-UMOUNT": https://github.com/Bs0Dd/MK90-projs/tree/master/GEN-UMOUNT

Программа совместима с программным обеспечением BASIC 1.0 и 2.0.


* Стандарт Genjitsu: https://github.com/vladkorotnev/SMPEmu/tree/master/pdp

* Модуль-эмулятор STMP: https://github.com/azya52/STMP

* Модуль-эмулятор PIMP: https://github.com/azya52/PIMP
