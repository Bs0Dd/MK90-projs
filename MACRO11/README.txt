                                * MACRO-11 для Linux * 
--------------------------------------------------------------------------------------
Данный микрокомплекс позволяет легко собирать код, написанный на макроассемблере DEC
MACRO-11, используя современный компьютер на базе ОС Linux.

Комплекс включает в себя адаптацию макроассемблера (macro11) от Йорга Хоппе
(программа собрана для платформы x86_64).

А также включает Perl-линковщик (obj2bin.pl) от Дона Норта.

Прилагается сборочный файл (makefile) для создания бинарных файлов, пригодных для
исполнения на микрокомпьютере Электроника МК90.

Микрокомплекс необходим для сборки остальных проектов в этом репозитории.
