                               * Тесты 2.0 для BASIC 1.0 *
-----------------------------------------------------------------------------------------

Системные тесты СМП и клавиатуры из ПО BASIC 2.0, адаптированные для запуска на
устройствах с ПО BASIC 1.0.

Данный образ размещается на СМП, который может быть вставлен в любой слот МК90.

После загрузки будет доступно меню с тремя опциями: тест СМП, тест клавиатуры и
возврат на загрузочный экран.

Тест клавиатуры из BASIC 2.0 несколько продвинут по сравнению с таковым в BASIC 1.0.
Помимо ASCII кода клавиши (в восьмеричной системе) показывается также код,
возвращаемый контроллером клавиатуры (комбинация столбца и строки). Присутствует
графическая индикация нажимаемой клавиши (и выдаваемого ей символа с учетом
модификаторов РУС/ЛАТ и ВР/НР).

Тест СМП в BASIC 1.0 является неработоспособным по причине ошибке в программе.
Поэтому данная адаптация из версии 2.0 является единственным способом произвести
тестирование модулей МПО-10 на МК90 с BASIC 1.0.
Основным отличием от исходного теста является автоматическое определение слота
с тестируемым модулем. Он всегда противоположен тому, с котого были загружены
тесты (т.е. если тесты загружены с СМП1, программа будет тестировать СМП0 и
наоборот). Такая особенность препятствует случайному стиранию СМП с тестами.
Перед началом теста необходимо указать начальный и конечный адрес памяти, в рамках
которых будет проводиться тестирование. Затем необходимо указать количество проходов,
т.е. повторений теста. Все числа указываются в восьмеричной системе.
Для полного тестирования модуля МПО-10 (10КБ) необходимо указать адреса 0 и 23777.
Если в тестируемом слоте модуль МПО-10 не установлен, программа будет указывать
на ошибку чтения памяти (считано: 377).

Тест ЖКИ в версии 2.0 отличается лишь наличием функции зацикленного теста
(поочередное переключение тестовых изображений), поэтому адаптирован не был.
